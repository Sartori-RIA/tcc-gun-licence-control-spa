# Pf.Gov.Br

Projeto gerado com [Angular CLI](https://github.com/angular/angular-cli) version 1.2.1.

## pré-requisitos 
+ **NodeJs >= 9.8.0**
+ **npm >= 5.7.1**
+ **Angular-CLI >= 1.7.3**

## Rodar a aplicação

+ **`npm install`**
+ **`ng serve` para executar o servidor em `localhost:4200`**
+ **Database MySQL `sinarm_db` criado**
+ **Iniciar a API Java**
